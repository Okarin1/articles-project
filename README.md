# reading主题模板


- 基于 [VuePress](https://vuepress.vuejs.org/)
- 主题 [vuepress-theme-reading](https://github.com/okarin1/vuepress-theme-reading)
- 自动部署 [vercel](https://vercel.com/)

## 克隆

```sh
git clone https://github.com/Okarin1/reading-project.git
```

## 安装依赖

```sh
yarn install
```

## 本地运行

```sh
year dev
```

## 打包

```sh
yarn run build
```
